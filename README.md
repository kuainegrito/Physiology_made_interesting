# Physiology Made Interesting

Making physiology intuitive, visual, interactive, and fun through mini-games, simulations, and educational visualizations.

---

## About This Project

**Physiology Made Interesting** is an open educational project that transforms complex physiology concepts into interactive experiences.

Instead of only reading static textbook diagrams, learners can:

* *feel* fatigue during repeated tapping,
* visualize ATP production and substrate utilization,
* understand lactate threshold dynamically,
* experience cardiovascular responses in real time,
* and explore physiology through gameplay.

The goal is to bridge:

* **physiology**
* **sports science**
* **medicine**
* **education**
* **visualization**
* and **game mechanics**

to create a more intuitive learning experience.

## Educational Philosophy

This project is based on the idea that:

> Physiology is not merely something to memorize — it is something to experience.

Many physiological concepts are difficult because they are:

* dynamic,
* multi-system,
* nonlinear,
* and highly time-dependent.

Interactive simulations help learners:

* build intuition,
* reduce cognitive load,
* connect theory with sensation,
* and improve long-term retention.
